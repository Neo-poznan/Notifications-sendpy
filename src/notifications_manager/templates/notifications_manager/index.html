Сервис для отправки нотификаций Notifications-sendpy

Внимательно ознакомьтесь с инструкцией по работе с сервисом прежде чем начинать работу

Как начать работу?

1. Пройдите регистрацию и войдите в аккаунт При регистрации используйте вашу корпоративную почту. Вы можете использовать свою личную почту, но в таком случае письма отправленные с этого сервиса будут попадать в спам. Из личных почт принимаются только яндекс и гугл почта.

2. Для следующих шагов необходимо будет настроить вашу электронную почту. Инструкция для яндекса актуальна и для корпоративной почты. Для этого нужно зайти с корпоративной почты.

Инструкция для яндекса

1. Перейдите в сервис "Яндекс.Почта" Затем "настройки", "Все настройки"

2. Перейдите в раздел "почтовые программы" и проставьте везде галочки в разделе "Разрешить доступ к почтовому ящику с помощью почтовых клиентов"

3. Далее "Безопасность", "Пароли приложений"

4. Создайте пароль приложения: придумайте ему название и сгенерируйте пароль. Внимание! Он отображается один раз, так что не забудьте его сохранить

Инструкция для gmail

1. Если у вас не включена двухэтапная аутентификация, то перейдите по этой ссылке и включите доступ ненадежных приложений

\

2. Если у вас включена двухэтапная аутентификация, то перейдите по той же ссылке и нажмите "Подробнее". Справа выскочит меню и в нем нужно будет нажать "Создайте пароль приложения" Далее просто следуйте указаниям google

Как отправить сообщения

1. Создайте рассылку в верхнем меню

2. Загрузите электронную таблицу с емейлами

3. Добавьте свой SMTP пароль (пароль приложений)

4. Добавьте настройки сервера. При отправке с личной почты(не рекомендуется) этот этап можно пропустить.

5. Сверстайте сообщение в редакторе или загрузите свою верстку, а также не забудьте добавить заголовок

6.. Нажмите кнопку "Отправить"
